# IP Addressing Scheme

This document outlines the IP addressing scheme used in the network.

## LAN

- **Sales (VLAN 10):** 192.168.10.0/24
- **HR (VLAN 20):** 192.168.20.0/24
- **Finance (VLAN 30):** 192.168.30.0/24
- **Admin (VLAN 40):** 192.168.40.0/24
- **ICT (VLAN 50):** 192.168.50.0/24
- **Servers (VLAN 60):** 192.168.60.0/24
- **Voice (VLAN 70):** 192.168.70.0/24
- **Wireless (VLAN 80):** 192.168.80.0/24
- **Management (VLAN 100):** 192.168.100.0/24

## WAN

- **ISP1:** Public IP Range 1
- **ISP2:** Public IP Range 2

## DMZ

- **DMZ (VLAN 90):** 10.10.10.0/24
