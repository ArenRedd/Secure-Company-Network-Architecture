# Security Policies

This document outlines the security policies for the network.

## Access Control

- **Principle of Least Privilege:** Users and devices should only have access to the resources necessary for their function.
- **Role-Based Access Control (RBAC):** Access rights will be assigned based on job roles.

## Device Hardening

- All network devices will be hardened by disabling unused services, changing default passwords, and implementing strong password policies.

## Monitoring

- The network will be monitored 24/7 for security incidents.
- All network traffic will be logged and audited.
