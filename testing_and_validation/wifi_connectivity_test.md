# WiFi Connectivity Test

This document describes the WiFi connectivity test.

## Test Plan

- [ ] **Corporate-WiFi:**
    - [ ] Connect to the "Corporate-WiFi" SSID with a corporate device.
    - [ ] Verify that the device gets an IP address from the correct VLAN.
    - [ ] Verify that the device can access internal resources.
- [ ] **Guest-WiFi:**
    - [ ] Connect to the "Guest-WiFi" SSID with a guest device.
    - [ ] Verify that the device gets an IP address from the guest VLAN.
    - [ ] Verify that the device can access the internet.
    - [ ] Verify that the device cannot access internal resources.
