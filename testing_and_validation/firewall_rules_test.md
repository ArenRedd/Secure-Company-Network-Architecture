# Firewall Rules Test

This document outlines the firewall rules test.

## Test Plan

- [ ] **Inside to Outside:** Verify that an internal user can access the internet.
- [ ] **Outside to Inside:** Verify that an external user cannot access an internal PC.
- [ ] **Outside to DMZ:** Verify that an external user can access the web server in the DMZ.
- [ ] **DMZ to Inside:** Verify that the web server in the DMZ cannot access an internal PC.
